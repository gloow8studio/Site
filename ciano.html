DevOps & Infra

Ferramentas Core

Docker: Containerização de aplicações e ambientes isolados.

Terraform: Infraestrutura como código — provisionamento na nuvem.

NGINX: Proxy reverso, load balancer e servidor web de alta performance.

Monitoramento & IAs

Datadog: Observabilidade completa — logs, métricas e traces.

New Relic: APM e análise de performance em produção.

GitHub Actions: CI/CD pipelines automatizados.

Mentalidade DevOps

💡 DICA: Aprenda isso cedo: deploy manual em VPS. Depois automatiza.

👉 Sequência: VPS manual → Docker Compose → CI/CD → Kubernetes

ComfyUI / Ollama: LLMs locais para automação com IA sem depender de APIs externas.

n8n: Mais potente que o Make para automações com perfil dev.

CrewAI / AutoGen: Agentes que trabalham em equipe (pesquisa, escreve, posta).
